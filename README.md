# OPA 數位教具開發與實務量測課程整合專案
本活動整合資訊開發與技職教育專長，利用 Flutter 打造跨平台互動電路模擬 App，引導學生透過平板進行數位探索，並結合 **OPA 反相與非反相放大器** 實習課程教案，帶領學生從軟體模擬平滑銜接至實體電路量測—，藉由操作示波器、訊號產生器等專業儀器與學習單，建構從虛擬模擬到硬體實踐的全方位學習路徑。
